Manufacturing 
PLC: Standing for “Programmable Logic Controller”, PLCs are considered the central “brains” of an automation which receives inputs and outputs and facilitates communication between separate parts. 

HMI: Standing for “Human-Machine Interface”, HMIs are generally visual representations of the processes that the PLC manages and represents the data collected by the PLC as-well, offering operators easy access to controlling the stations.
  
LGV: Standing for “Laser-Guided Vehicle”, LGVs are automatic vehicles which navigate the facility using laser pointers and reflective components.
  
FMEA: Standing for "Failure Mode and Effects Analysis”, is a strategy used to identify potential problems and situations which would compromise the effectiveness or safety of a product or process.
  
PDM: Standing for “Product Data Management”, a PDM is a system which can be integrated with systems, such as CAD software, and be used to store, manage and organize data for products.
  
Device Tree: A hierarchical data structure that contains several hardware components with descriptions from the OS (operating system).
  
ERP: Standing for “Enterprise Resource Planning”, an ERP is a software system which helps organizations and businesses manage data  and processes by automation, while enabling data sharing and collaboration between departments.
  
Bottleneck: A strain or stoppage on output caused by a considerable loss of speed concerning the flow of finished subassemblies.
  
End-Effectors: Also known as end-of-arm tools, are specialized tools located at the end of a robot arm, allowing the robot to perform a certain task accordingly.
  
PROFINET: An industrial ethernet communication standard commonly utilized within manufacturing to allow communication between controllers such as the PLC and devices including, sensors, drives, and robots.

SCADA: Standing for “Supervisory Control and Data Acquisition”, a SCADA is a system which is used to monitor and control industrial processes and systems.

SQL: Standing for “Structured Query Language”, SQL is a programming language used to manage and manipulate relational databases.

Stacklights: A visual indicator used to show the status of a machine or process.
